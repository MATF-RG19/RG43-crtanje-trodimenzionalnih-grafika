# RG43-GraphingCalculator3d

Controls:

| Key           | Command        |
| ------------- | -------------- |
| w             | rotate up      |
| s             | rotate down    |
| a             | rotate left    |
| d             | rotate right   |
| z             | zoom in        |
| x             | zoom out       |
| g             | grid on/off    |
| k             | switch plotter |
| n             | next plot      |
